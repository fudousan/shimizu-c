施工事例ページでは10枚の画像を一覧表示しますが、そのまま縦に長く並べるだけでは閲覧しづらくなります。CSSを用いてギャラリー風のレイアウトに整えましょう。ポイントは以下の通りです。

  • 複数列レイアウト: 画面が十分広い場合は1行に複数の画像を並べ、横にレイアウトします。例えばPCでは3列や4列格子状に配置し、スマホなど狭い画面では列数を減らして1 列か2列表示に切り替えます。こうすることでスペースを有効活用し、かつ各画像が小さくなりすぎず視認性を保てます。

CSSコード例 (style.cssに追加):

css
/* 施工事例ギャラリーのレイアウト */
.works-gallery {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(250px, 1fr));
  gap: 16px;  /* グリッドの隙間を16pxに設定 */
  padding: 20px 0;
}
.works-gallery figure {
  margin: 0;
}
.works-gallery img {
  width: 100%;
  height: auto;
  display: block;
  border-radius: 4px;  /* 角を少し丸める等お好みで */
}
.works-gallery figcaption {
  margin-top: 8px;
  font-size: 0.9em;
  text-align: center;
  color: #555;  /* キャプションの文字色（薄めのグレー等） */
}

    。たとえば幅の広い画面では1行に4つ程度並び、幅が狭くなるにつれて3列、2列と減り、最終的にモバイルでは1列になる、といった挙動になります（メディアクエリ無しで実現できます）。

  • gap: 16px;はグリッドアイテム（各figureブロック）間の間隔を設定しています。16pxの余白を設けることで、画像どうしがくっつかず見やすくなります。

  • .works-gallery figcaptionではキャプション文字に上マージンを少し付けて間隔を空け、文字サイズをやや小さく、中央寄せにしています。色も少し薄いトーンにすることで画像の目立ちを損なわないようにしています。
